Resources

Intro

Very different things have been so helpful when learning how to code. You can find some of these listed and explored in depth down below.

  • Learning code and how to code
  • Layout and looks
  • Extras

Learning code and how to code

  • Codecademy has been invaluable in learning the basics and getting the feel for new languages.
  • Youtube is a wonderful place to get additional insight on things you don't understand. The search bar is your best friend. Here's a video on CSS (Cascading Style Sheets) that has helped me.
  • Opportunity College has been absolutely amazing and instrumental in my progress thus far. If you're looking for somewhere to learn with amazing people, consider applying; and if not, consider reaching out to see if you can give back/ help in some way.

Layout and looks

  • Wireframes like Claritee are really helpful for planning the layout of your website
  • Color is a very important component of the look of your website. If you can't choose, try a color generator like coolors.co
  • The font is another very important component of the look of your website. If you can't decide or want to see how different fonts look together, try fontjoy.com or use Google fonts to browse.
  • For visulas and pictures that are free and available to use for commercial or non-commercial reasons, check out Unsplash

Extras

  • Music helps me get in the groove when coding. It also helps when decompressing from a particularly difficult piece of code. My streaming site of choice: Spotify
  • Planning out my time so I know when I have time to learn coding, or time to work on a project is really helpful. I like Google Calendar for this

Thanks for taking the time to look through. I hope to update this as I continue learning - come check out the updates every once in a while. Find me on Github!
